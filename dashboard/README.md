# Dashboard — Рынок керамической плитки России

Интерактивный дашборд на Streamlit для анализа данных о керамической плитке, собранных с 4 российских магазинов.

## Магазины в данных

| Магазин | Остатки | Онлайн-заказ |
|---------|---------|--------------|
| LemanaPRO | ✅ | ✅ |
| OBI | ✅ | — |
| Petrovich | ✅ | — |
| Keramogranit_ru | — | — |

## Установка и запуск

```bash
pip install -r requirements.txt
streamlit run dashboard.py
```

Дашборд запустится по адресу `http://localhost:8501`

## Разделы дашборда

- **Обзор рынка** — KPI: количество позиций, брендов, средняя и медианная цены
- **Распределение цен** — box plot и гистограмма по магазинам
- **Структура ассортимента** — доли по магазинам, материалам, типам поверхности
- **Бренды** — топ брендов по количеству позиций и средней цене
- **Форматы и дизайн** — популярные форматы и дизайны
- **Страны производства** — количество позиций по странам
- **Остатки** — распределение остатков по магазинам и материалам
- **Таблица данных** — с возможностью сортировки и фильтрации

## Фильтры

- Магазин (мультивыбор)
- Материал (Керамогранит / Керамика / Клинкер)
- Тип поверхности
- Диапазон цен
- Только с ценой / только с остатками
