# Документация проекта Scraping Russia

## Структура папки docs

```
docs/
├── screenshots/          # Скриншоты дашбордов и визуализаций
│   ├── dashboard_*.png  # Дашборды Power BI
│   ├── data_*.png       # Примеры структуры данных
│   └── charts_*.png     # Графики и диаграммы
│
└── examples/            # Примеры данных
    └── sample_data.json # Образец данных (10-20 записей)
```

## Рекомендуемые скриншоты

### 1. Дашборды Power BI
- Общий обзор рынка
- Анализ цен по брендам
- Распределение по форматам
- Карта доступности товаров
- Анализ скидок

### 2. Примеры данных
- Структура JSON файла
- Пример карточки товара
- Таблица с несколькими записями

### 3. Процесс работы
- Запуск парсера (опционально)
- Результаты выполнения
- Лог обработки данных

## Добавление скриншотов

1. Сделайте скриншот
2. Сохраните в папку `docs/screenshots/`
3. Используйте понятные имена: `dashboard_overview.png`, `data_structure.png`
4. Добавьте в Git:
   ```bash
   git add docs/screenshots/*.png
   git commit -m "Add screenshots for documentation"
   git push
   ```
5. Обновите главный README.md, добавив ссылки на скриншоты
